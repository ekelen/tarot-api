Rider-Waite-Smith Cards API

Using data from The Pictorial Key to the Tarot by Arthur Edward Waite.

View YAML documentation.

Examples:

Get all cards: https://rws-cards-api.herokuapp.com/api/v1/cards

Search all cards with "difficult" in reversed or upright meanings: https://rws-cards-api.herokuapp.com/api/v1/cards/search?meaning=difficult
